# Buscador de Usuarios - Next.js

Aplicación para buscar y visualizar usuarios utilizando Next.js 14, TypeScript, Material-UI y la API de JSONPlaceholder.

# Como ejecutar el proyecto

- Tener instalado node 22.20.0 o superior
- Node.js 18+ 
- npm o yarn

# Instalación

- cd .\nextjsproject\
- npm install

# Uso
- npm run dev
- abrir en el navegador con http://localhost:3000

# Testing

- npm run build
- npm run test

# Tecnologías utilizadas

- Next.js 14 
- TypeScript 
- Material-UI
- Vitest
- Testing Library

# Cobertura de Tests

- Componentes de UI
- Llamadas a la API
- Navegación entre páginas
